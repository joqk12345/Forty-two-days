# 《42天》语言风格与场景写作规范

## 一、总体语气

《42天》的语言应当克制、冷静、具体。不要写成热血历史评书，也不要写成心学讲义。叙述者不替人物喊口号，只把场景、动作、物件和选择摆出来，让读者自己感到压力。

本项目的总写作规则见 `42-days-project-rules.md`。风格上尤其遵守三点：局势问题先行，机制先于评价，条件必须跟着判断走。

目标气质：

- 冷开场
- 短句推进
- 物件承压
- 情绪后置
- 因果清楚
- 余味复杂

一句话：少解释，多让东西发生。

## 二、句子风格

### 1. 多用短句

适合：

```text
门闩落下。
乐声还在。
孙燧没有回头。
```

不适合：

```text
随着沉重的门闩声响起，整个王府宴厅里的气氛顿时发生了微妙而令人窒息的变化。
```

短句可以制造压迫。长句只在需要展示复杂推理时使用，比如王阳明推演三策。

### 2. 先动作，后意义

不直接写：

```text
王阳明非常冷静。
```

改写成：

```text
王守仁把湿信摊平，先用镇纸压住缺了半边的地方，又让人取江西图。
```

读者会从动作里读出冷静。

### 3. 少用抽象词

少用：

- 正义
- 命运
- 历史
- 人心
- 大势
- 天下兴亡

这些词不是不能用，而是不能频繁裸露。它们应当藏在物件和动作里。

比如“天下”可以落到：

- 一枚印
- 一封檄文
- 一座城门
- 一条江
- 一面旗

## 三、场景写法

每个场景必须回答四个问题：

1. 谁在场？
2. 他想要什么？
3. 什么阻止他？
4. 场景结束后，局势有什么变化？

如果一个场景没有变化，就删掉或并入别章。

## 四、章节结构

每章建议采用四段式：

### 1. 进入场景

用地点、天气、物件开场，不用解释开场。

示例：

```text
丰城驿舍的灯，一夜没灭。
```

### 2. 建立冲突

让人物面对一个必须处理的问题。

示例：

```text
第二封信比第一封完整。也更坏。
```

### 3. 做出选择

人物必须行动，不要只讨论。

示例：

```text
王守仁把“救安庆”四个字划掉。
```

### 4. 留下钩子

结尾落在动作、物件、消息上，不用总结。

示例：

```text
那枚棋子没有留在南京，最后被他压回南昌。
```

## 五、人物语言

### 王阳明

说话短，少解释。重要判断常用问题引出。

适合：

```text
“若他不取南京，守江西，能守几日？”
“若他取南京，天下几日能知？”
“先定名。”
```

不适合：

```text
“我们现在必须从底层战略逻辑上打断他的叛乱升级链条。”
```

现代分析可以留在创作笔记里，不要进入古人对白。

### 朱宸濠

说话要有威压，不能一直暴怒。他越危险，越不必大声。

适合：

```text
“孙公，密旨在此。”
“你不看旨，是不认本王，还是不认太后？”
```

不适合：

```text
“哈哈哈哈，今日就是本王夺取天下的开始！”
```

### 伍文定

直接、务实、敢问。

适合：

```text
“安庆若破，罪算谁的？”
“船够，粮不够。”
```

### 普通人

普通人不要替作者讲道理。他们只说自己看见的、怕的、需要的。

适合：

```text
“官爷，船烧了，我一家吃什么？”
```

## 六、物件使用原则

每个重要物件至少出现三次：

1. 第一次：作为普通物件出现。
2. 第二次：暴露重要性。
3. 第三次：完成情节或主题回收。

示例：酒杯

- 第1天：王府宴厅中盛酒。
- 第2天：酒杯旁有血。
- 第41天：同样的杯子被拿来盛粥。

这样权势崩塌不需要解释，杯子的用途变化就够了。

## 七、战争写法

不要写成战报：

```text
双方大战，宁王军大败。
```

要写成身体经验：

```text
火从船腹里窜出来，先烧绳，再烧帆。有人跳水，甲叶一沉，人就不见了。
```

战争场面重点写：

- 声音：鼓、喊、木裂、火爆。
- 触感：湿甲、烫绳、滑船板。
- 视觉：烟、旗、火、血水。
- 失控：命令传不到、船掉不了头、火回烧。

## 八、心学写法

心学不能写成课堂。不要让王阳明长篇解释“知行合一”。

要写成选择：

- 知道宁王中策最危险，于是传檄拖慢。
- 知道安庆急，却仍攻南昌。
- 知道胜后有险，却仍如实写捷报。

心学落在行动里，才有小说力量。

## 九、悬疑写法

这部小说的悬疑不是“凶手是谁”，而是：

- 宁王下一步走哪一策？
- 地方官会不会响应？
- 王阳明的疑兵会不会被识破？
- 不救安庆会不会造成大败？
- 攻南昌能不能成功？
- 火攻会不会回烧？
- 胜利会不会被改写？

每一幕都要有一个主要悬疑问题。

## 十、从《迟到》借鉴到《42天》的落点

### 1. 冷开场

《迟到》先写雪夜埋尸。《42天》先写王府落闩。不解释，先让读者进危险现场。

### 2. 短章

《迟到》单章短，推进快。《42天》每章一天，不追求大段历史说明。

### 3. 物件推进

《迟到》用手机、纸、账、车。《42天》用檄文、印信、地图、酒杯、船、火。

### 4. 情绪克制

不喊悲壮，不喊热血。孙燧不跪，王阳明划字，朱宸濠改令，这些动作比评价更重。

### 5. 制度压迫

《迟到》写流程、卷宗、通报。《42天》写印信、密旨、檄文、奏报、亲征、记功。权力不是抽象词，是一套文书和仪式。

## 十一、禁忌清单

- 不要把王阳明写成预言家。
- 不要把朱宸濠写成傻反派。
- 不要让人物说现代分析词。
- 不要用大段旁白解释心学。
- 不要让战争只有地图推演，没有身体代价。
- 不要把普通人写成只会惊呼的背景。
- 不要每章都用总结句结尾。
- 不要让结尾过度圆满。

## 十二、推荐章节语感

每章开头可以这样：

```text
南昌王府的灯，比往常多点了三重。
```

```text
丰城驿舍的雨，落到后半夜还没停。
```

```text
安庆来的信使，是被人抬进来的。
```

```text
湖上的风，午后转了。
```

每章结尾可以这样：

```text
门闩落下以后，乐声还响了半支曲子。
```

```text
那枚印盖下去，县令的手才开始抖。
```

```text
王守仁把棋子从南京拿起，压回南昌。
```

```text
湖面最后一点火，到了天亮才灭。
```
