# 攻复南昌的城门与攻城方式

## 核验结论

王守仁奏疏对攻城部署记载明确：官军先破城外伏兵，再分攻七门，以梯绳登城。现无可靠史料支持“水门内应潜入—夺粮仓—开西门”为破城主因。

| 日期 | 行动 | 等级 |
|---|---|---|
| 七月十八日 | 至丰城，分配七门；遣刘守绪等四百人夜袭新旧坟厂千余伏兵 | A |
| 七月十九日 | 发市汊，大誓各军，薄暮齐发 | A |
| 七月二十日黎明 | 各军抵指定城门，四面呼噪并进，以梯绳登城 | A |
| 城破后 | 擒宜春王拱樤、内官万锐等千余；救火、释胁从、封府库、收印信 | A |

| 顺化门 | 赣州知府邢珣 |
| 惠民门 | 袁州知府徐琏 |
| 永和门 | 临江知府戴德孺 |
| 章江门 | 通判胡尧元、童琦 |
| 德胜门 | 新淦知县李美 |
| 进贤门 | 都指挥余恩 |

其他官兵“乘七门之衅，傍夹攻击，以佐其势”。《明史》另记伍文定夜半抵广润门，守兵惊散；次日黎明诸军梯绳登城。

## 正文边界

- 七门分攻、外围伏兵、新旧坟厂、梯绳登城属于应守硬史实。
- 城内小吏、乐工传递消息可以保留为艺术加工，但只能帮助动摇守军、破坏号令或提供城防情报。
- 水门行动不得继续承担攻破南昌的决定性因果，也不宜写成大队官军由水门入城。
- 可写小股人员尝试接应、制造混乱；最终城破必须落到七门强攻和守军溃散。
