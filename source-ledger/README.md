# 《42天》史料台账

本目录记录正文中高风险史实的来源、证据等级和艺术加工边界。

## 使用规则

- A：实录、奏疏、年谱、文集等原始或近原始材料。
- B：官修史书、可靠校注本或现代学术研究。
- C：通俗文章、讲座转写、百科及未核验笔记，只作检索线索。
- 正文不放引文；每次涉及精确日期、官职、路线、数字、战术和人物结局时，在这里留下可复查记录。
- 原始材料若带有道德评断、传闻或互相矛盾的细节，事实层与评价层分开记录。
