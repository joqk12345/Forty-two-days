# 五项史实核验后的正文修改单

## 已可直接修正

1. 第39章“湖口接战”改为“黄家渡接战”。
2. 第39章官职落款改为“巡抚南赣、提督军务右佥都御史王守仁”。
3. 第39至42章删除“一正两副、三路分送、回票确认”，改成七月三十日两份内容不同的捷音疏：
   - 《江西捷音疏》，千户詹明亲齎；
   - 《擒获宸濠捷音疏》，千户王佐亲齎。
4. 火攻奏报不写未经史料确认的具体风向，只写二十五日“风势不便”、二十六日“火及副舟”。

以上四项已在正文完成。

## 南昌线整体重构（已完成）

旧稿第16至24章曾以“水门内应—八十余人潜入—夺粮仓—开西小门、东门—擒万锐”为破城主链。可靠史料的主链则是：

> 七月十八日分攻七门 → 刘守绪四百人夜破新旧坟厂伏兵 → 十九日市汊誓师、薄暮齐发 → 二十日黎明七门同时附城 → 梯绳登城 → 守军倒戈溃散。

若只在旧稿中插入“七门齐攻”，伍文定会同时身在水门内和广润门，形成新的空间冲突。因此已按以下方式整体改造：

| 章节 | 保留功能 | 应改历史动作 |
|---|---|---|
| 第16章 | 决定攻根本、城内消息线 | 城图改列七门；水门仅作传递消息和侦察的小闸，不是破城入口 |
| 第17章 | 征船、船户与物资 | 船只用于运输、侦察和后续湖战准备，不为九十人潜入服务 |
| 第18章 | 军议与退路 | 明定七门分工、坟厂伏兵、梯绳和市汊誓师；水门情报为辅助 |
| 第19章 | 夜行压力 | 改为刘守绪四百人间道袭坟厂，以及七哨薄暮向各门推进 |
| 第20章 | 阿衡、陈槐城内视角 | 传入宁王主力东出、城外伏兵败退、七门同时有敌等消息；可保留破坏鼓号、藏匿册印 |
| 第21章 | 高廷槐正式站队 | 县印、断粮、回文线可基本保留；南昌部分改为坟厂败卒入城、守军调动 |
| 第22章 | 城内秩序先乱 | 不再夺粮仓开门；改为守军争抢城防物资、仓丁拒绝搬粮、万锐分兵失措 |
| 第23章 | 夺城高潮 | 伍文定在广润门；其余六门并攻，梯绳登城。阿衡等只负责藏人、指路、救火或保护册印 |
| 第24章 | 朱宸濠得知根本已失 | 急报明确写“坟厂败、七门并攻、梯绳登城、拱樤万锐被擒” |

第16至24章及第28章相关回指已按上表修改完成。

## 安庆线修改点

1. 第15章应点出至少一名真实守将，优先杨锐；张文锦、崔文可随场景加入。
2. “泼火油”改为投掷浸油燃烧的苇束。
3. 可加入史料明确的云楼、飞楼、巨型天梯、沸汤、民兵及妇女运石。
4. “炮打出裂口”无直接证据，宜删或降为人物误判。
5. 负伤信使送“安庆知府急报”属虚构；若保留，应避免写成唯一可靠消息。王守仁奏疏可确认的是谍报与安庆逃回船户。

## 水路与战后时间线边界

- 可写东下顺流、回师逆水；不可把具体浅滩、风向写成已核史实。
- 第39至42章只能处理七月三十日前后的初次捷音和亲征消息。
- “奉大将军钧帖重上捷音、把随驾权贵写入功劳”发生在正德十五年七月，须另设时间跳跃，不能留在四十二天即时线中。
- 九月十一日王守仁才自南昌亲自押俘启程；杭州交张永的确日仍待核。
